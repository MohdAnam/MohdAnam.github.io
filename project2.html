Hindi_Varnamala

An Augmented Reality App which is based on Hindi Varnamala and shows 3D objects related to each letter.

Children today can be taught better with immersive technology that is available, namely "Augmented Reality".

This app helps children to use tablets or mobile phones on markers where objects emerge from the markers as children place the camera on it.

It creates a more engaging learning atmosphere. With such an ecosystem, students become more engaged in their learning, which will ultimately boost the number of students in school.

It will help students understand the shape and size of different things more effectively and quickly.

Features

There are two basic features implemented on the 3D models - scaling and rotation of the 3D models using touch gestures.

Scanning the custom cube markers. Scaling the 3D model using touch gestures.
Scanning another marker. Rotating the 3D model using touch gestures.
