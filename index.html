Projects

Project 1: Alphabet Black_White

I developed an Augmented Reality application which focuses on augmenting 3D models by scanning related to alphabet marker. It creates a more engaging learning experience, and with such an ecosystem, students become more engaged in their learning that will ultimately boost the number of students in school.

Project 2: Varnmala

I developed a marker-based AR app centered around the Hindi Varnmala alphabet. By scanning related markers, it augments 3D models for a captivating learning experience. This app encourages active student engagement, leading to increased enrollment and enhanced learning outcomes.

Project 3: Multimodel Alphabet

I designed an innovative AR app that enhances learning through augmented reality by displaying multiple 3D models of objects associated with a specific letter. For instance, when scanning the letter "A," a 3D model of an airplane appears on the screen. This interactive experience is facilitated through tablet scanning of markers, making learning engaging and effective. The app not only supports educators in their teaching process but also creates a stimulating learning environment, encouraging students' active participation and interest in their studies.

Project 4: Test Animal AR

The Augmented Reality Animal Interaction App is an innovative and engaging application that brings a variety of 3D animals to life through augmented reality technology. Users can interact with 12 different animals by pointing their device's camera at specific image targets, providing an immersive experience with realistic animations and behaviors.

Project 5: Numeric AR App

The Augmented Reality Numeric Interaction App is an innovative application that harnesses the power of augmented reality to facilitate interactive learning and creativity. This app enables users to create two-digit and one-digit numbers using 3D models corresponding to numeric digits. Users achieve this by placing two different image targets side by side, which combine to form a two-digit number, or by using a single image target for one-digit numbers.

Project 6: AR Birthday Card

The "AR Birthday Card" is an augmented reality (AR) application that transforms traditional greeting cards into interactive and memorable experiences. This project showcases expertise in marker-based AR technology, allowing users to augment photos, videos, and 3D models, such as birthday cakes, onto physical birthday cards. It introduces a creative and personalized way to convey heartfelt wishes and celebrate special occasions.

Project 7: AR facts information with virtual buttons

The "AR Facts Information with Virtual Buttons" is an innovative augmented reality (AR) application designed to provide an immersive and interactive learning experience. This project showcases skills in developing AR applications that combine virtual buttons with informative video content. It offers users an engaging way to access educational information by simply interacting with virtual elements in the real world.

Project 8: AR Elephant Model

The "AR Elephant Model" is a captivating augmented reality (AR) application that demonstrates creative and technical skills in AR development. This project features an animated elephant that interacts with its environment based on user input.The elephant follows a predefined path, offering an engaging and visually appealing experience.

Project 9: AR Tap Object

AR Tap Object" is an exciting augmented reality (AR) application that demonstrates expertise in marker-less AR technology. This project offers a straightforward yet engaging user experience. When users launch the app, it scans the environment for a suitable plane and then allows them to interact with a 3D capsule model by tapping on the screen. The model's position changes in real-time, creating an interactive and dynamic AR experience.
